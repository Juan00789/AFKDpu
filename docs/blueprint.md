# **App Name**: Ephemeral Connect

## Core Features:

- User Registration/Login: User authentication and role selection (client, provider, employee).
- User Dashboard: Dashboard to manage connections, tasks, and communications, featuring a sidebar for navigation and a main section for displaying active connections and tasks.
- Temporary Chat: Real-time chat with temporary messages that disappear after a set time or connection state change.
- Smart Prompts: AI-powered tool which generates dynamic suggestions for connection updates, offering users helpful nudges and ideas based on the current state of the relationship.
- Rule Configuration: Ability to set personalized rules for each connection, triggering actions based on connection status (e.g., sending notifications when a connection becomes 'Fading').
- Contextual Notifications: Real-time contextual notifications to inform users of state changes, updates, or recommended actions for their connections.
- User Profiles: Individual profile pages displaying user avatars, basic information, connection history, and ratings/comments for providers.

## Style Guidelines:

- Primary color: Vibrant purple (#A020F0) to evoke creativity and connection.
- Background color: Light lavender (#E6E6FA), a muted tint of the primary, for a calm, neutral backdrop.
- Accent color: Deep blue (#00008B), to offer a contrasting complement that suggests importance and stability.
- Body font: 'Inter', sans-serif. Headline font: 'Space Grotesk', sans-serif. To offer a balance of legibility with a techy touch.
- Use minimalist, abstract icons to represent connection states and emotions, in a consistent visual language.
- Dashboard layout designed to prioritize active connections and tasks, with a clear visual hierarchy to guide user attention.
- Subtle animations on state transitions (e.g., when a connection changes from 'Neutral' to 'Vibrant') to provide feedback and enhance engagement.
